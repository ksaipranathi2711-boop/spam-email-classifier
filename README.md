# Spam Email Detector

## Overview

This project is a Machine Learning based Spam Email Detector built using Python, Flask, and Scikit-Learn.

The model analyzes email or message content and predicts whether it is:

* Spam
* Ham (Not Spam)

## Technologies Used

* Python
* Flask
* Scikit-Learn
* Pandas
* HTML/CSS

## Features

* User-friendly web interface
* Real-time spam prediction
* Machine Learning classification
* Lightweight and fast

## How to Run

1. Train the model

```bash
python train_model.py
```

2. Start the application

```bash
python app.py
```

3. Open

http://127.0.0.1:5000
